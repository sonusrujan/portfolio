# **App Name**: Apex Portfolio

## Core Features:

- Hero Section Parallax: Implement a smooth, scroll-based parallax effect for the hero section using the provided WebP video link (https://drive.google.com/file/d/1DPk3a4boLsVdcynvXWnWD2_XiuODadG/view?usp=sharing).
- Interactive Project Showcase: Create an interactive grid or stack of project cards with smooth popup transitions to display project details.
- Resume Analysis: Automatically extracts skills from a resume to display on a section using resume content in latex format.
- Contact Form Submission: Implement a simple and elegant contact/footer section with links to email, GitHub, and LinkedIn.
- Smooth Transitions: Integrate subtle, Apple-like animations and transitions throughout the website.

## Style Guidelines:

- Primary color: Subtle blue (#5271FF), evoking confidence and modernity.
- Background color: Off-white (#F0F2F5) for a clean and spacious feel.
- Accent color: Light Gray (#A9A9A9) for UI elements and subtle highlights.
- Body and headline font: 'Inter' sans-serif for a modern, objective feel. Note: currently only Google Fonts are supported.
- Use minimalist icons that subtly enhance key sections without clutter.
- Implement a clean, minimal layout with plenty of whitespace, drawing inspiration from Apple's design aesthetic.
- Apply smooth, physics-based animations (like fade-ins, parallax) triggered by scrolling and micro-interactions on hover.
